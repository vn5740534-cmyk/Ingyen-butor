# Ingyen butor

Premium, többnyelvű (HU/DE/EN) webapp váz, amely ingyenes bútorhirdetéseket jelenít meg.

## Fő funkciók
- HU/DE/EN nyelvválasztó
- Város + km szűrő
- Kártyás hirdetéslista képpel + leírással
- Modern, komoly vizuál

## Futás (lokálisan)
Egyszerű statikus webapp, nincs build lépés.
- Nyisd meg az `index.html` fájlt a böngészőben

## Deploy Vercel-re
1. Vercel → New Project
2. Válaszd ki ezt a repót
3. Deploy

## Adatforrás
A jelenlegi adatok demó jellegűek. Külső adatforrás csak hivatalosan engedélyezett módon integrálható.
